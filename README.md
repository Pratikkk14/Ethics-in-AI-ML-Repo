# Ethics-in-AI-ML-Repo
This repo has collections of experiments performed for practicing various ethics concepts that have to be used in modern ML models.

| Sr. No. | Title | Description |
|---|---|---|
| 4 | [Sentiment Analysis](Ethics_In_AI_Experiment_4.ipynb) | Analyzes audience sentiment on YouTube comments with a rule-based NLP pipeline (VADER), then summarizes polarity and engagement metrics to discuss model-driven interpretation and ethical implications. |
| 5 | [Earthquake Severity XAI(Explainable AI)](Ethics_Lab_5.ipynb) | Builds a Random Forest black-box model to classify earthquake severity (low vs high) and applies SHAP and LIME to explain global feature importance and single-prediction reasoning for ethical transparency.
